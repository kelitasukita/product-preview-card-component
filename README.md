## Setup

* Make sure that your installed Node version is compatible with the one described in package.json.


1. Clone the repo;


2. Go to the product-preview-card-component folder:


3. Install dependencies
<pre> npm install </pre>

4. Run project with: 
<pre> npm run dev </pre>
